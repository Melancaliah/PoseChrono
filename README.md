This tool was originally conceived as a plugin for **Eagle.cool**. However, to make it more accessible, a standalone Windows version is now available.

### Key Features

- **Format Versatility** - Full support for images, GIFs, and videos (.mp4) for moving or slow-motion models.
- **Training Modes** - Multiple modes tailored to your needs, including random or manual sorting.
- **Artist Toolset** - Silhouette view, B&W, Blurs, and custom grids/guides to check plumb lines.
- **Drawing Module** - Integrated light table feature to scribble directly over your references.
- **Proportion mesurement** - Built-in measuring tools (set units, compare lengths, and report measurements).
- **Timeline** - Comprehensive history to visualize your evolution and find practiced poses.

## Eagle (Manual installation)

If you need to install it manually, follow these steps:

1. **Create Placeholder:** In Eagle, go to `Plugins` > `Developer Options` > `Create Plugin`. Select **"window"** and name it `PoseChrono`.
2. **Open Directory:** Open the folder Eagle just created.
3. **Deploy Files:** Download the [Latest Releases](https://github.com/Melancaliah/PoseChrono/releases/latest) Download the `posechrono-eagle-<release>.zip` file and copy all files into the folder **EXCEPT** `manifest.json`.
4. **Update Manifest:** Copy settings (description, keywords...) from the release manifest to your local one. **Do not touch the `"id"` field.**
5. **Finalize:** Restart Eagle and start drawing!

## Windows (Version Standalone)

1. Go to [Latest Releases](https://github.com/Melancaliah/PoseChrono/releases/latest).
2. Download the `posechrono-desktop-<release>-setup.exe` file.
3. Launch the application.
